# JUROX OS

JUROX OS is the future operating system direction for JGS.

## Vision

JUROX is planned as an OS and runtime centered on Haicode, similar to how
Unix/Linux history is deeply connected to C.

## Early Concepts

- Kernel and user space mapping.
- Syscall and ABI mapping.
- Filesystem, process, memory, network, and permission models.
- Portable runtime forms for USB, SSD, HDD, and existing platforms.
