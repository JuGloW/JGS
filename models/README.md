# Models

This folder documents model strategy for Haijun and JGS.

## Rules

- Model weights are server-only.
- Do not commit `.gguf`, `.safetensors`, `.bin`, `.pt`, `.pth`, `.onnx`, or
  checkpoint files.
- Base models are selected by role, not by hype.
- Training starts only after owner approval and budget/time cap.

## Early GPU Target

- Provider: RunPod
- GPU: RTX A5000
- Purpose: first-run inspection, then small experiments if approved.
