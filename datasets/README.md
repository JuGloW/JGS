# Datasets

This folder documents dataset rules for JGS and Haijun.

## Rules

- Do not commit private raw data.
- Do not commit secrets.
- Do not commit large datasets before storage policy is ready.
- Curated examples should be small, reviewed, and traceable.

## Current Haijun Dataset Status

- Initial curated examples: 77
- Eval items: 20
- Location remains in the local Haijun package until a safe publishing policy is
  decided.
