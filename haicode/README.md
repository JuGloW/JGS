# Haicode

Haicode is the language center of the JGS ecosystem.

## Core Direction

- `hai*` is the language family.
- `haic*` is the foundational bridge for C/C++ and Unix/Linux concepts.
- Converhai maps other languages into the Haicode family while preserving
  behavior.

## Early Focus

- Syntax doctrine.
- Conversion rules.
- Behavior preservation.
- Toolchain integration with JCH.
- ABI direction with HCB.
