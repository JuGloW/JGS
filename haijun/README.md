# Haijun AI

Haijun is the planned private AI developer agent for the JuGloW JGS ecosystem.

## Role

- Assist JGS development.
- Analyze code and errors.
- Maintain JGS knowledge and memory.
- Help with Haicode, Converhai, JCH, HCB, and JUROX planning.
- Protect owner identity and confidential data.

## Server Policy

- Local computer is control-only.
- Model downloads happen only on the remote server.
- GPU is used only for approved training/evaluation sessions.
- No model weights are committed to GitHub.
